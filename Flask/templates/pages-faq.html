Team Oriented Questions
1. What is the main goal of team 28?

The goal of team 28 is to create a bot that is capable of trading stocks & cryptocurrency. The team is hoping to package this bot in an intuitive user interface.

2. How does team 28 define success?

The success of the team is based on how well they work together to create the most effective bot possible.

3. Why did team 28 choose to make a bot for the senior project?

Every member of team 28 has a strong interest in either finances or machine learning. Creating a bot that trades financial assets seemed to be an appropriate combination of the two fields.

Code Base Questions

Python is the main programming language used to create the project. Team 28 though it would be a good choice due to its simplicity and its great access to libraries and frameworks for machine learning applications.

The web application for the bot was created using Flask. Flask allowed team 28 to seamleassly combine the power of Python with the web-compatability of HTML and Javascript.

The trading strategies for the bot are created by team 28 using Python. Implementing the trading strategies into the bot was made possible by a framework named Blankly. Blankly allows team 28 to streamline the process of developing trading models and a backtesting engine and focus on creating the trading strategies alone.

TradingView is a cloud-based charting/analytical software that allows for stock/crypto info to be embedded into web pages. Team 28 uses TradingView to display stock history, live information, and future predictions from our machine learning models.
